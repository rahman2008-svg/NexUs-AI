# 🎹 Virtual Piano Skill

A smooth, responsive and horizontally scrollable virtual piano keyboard powered by **Web Audio API**.

This skill allows users to play music directly inside the NexUs AI environment.

---

## 📁 Files Overview

- **index.html**  
  The main entry point that loads the virtual piano interface.

- **index.js**  
  Handles logic and returns the webview URL that renders the piano UI hosted on GitHub.

---

## 🎯 Supported Prompts / Triggers

You can activate this skill using natural language such as:

- “Open virtual piano”
- “Play the piano”
- “I want to play piano”
- “Show me a piano keyboard”
- “Launch piano mode”

---

## ⚙️ Instructions

When triggered, the system should:

1. Call the `run_js` tool  
2. Load the virtual piano webview interface  
3. Render an interactive keyboard for user input  
4. Enable audio playback using Web Audio API
