# 🎧 Mood Music Skill (NexUs AI)

Skill for NexUs AI Gallery to generate music based on user mood using the **Loudly API**.

---

## 🎼 Overview

This skill allows NexUs AI to generate mood-based music by analyzing user input and selecting suitable genres via the Loudly API.

---

## 🔑 Loudly API Setup

Loudly API key can be obtained by creating an account:

👉 https://www.loudly.com  

Then generate your developer API key here:

👉 https://www.loudly.com/developers/apps
