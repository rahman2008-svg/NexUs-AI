# Nexus AI ✨

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](LICENSE)

**Explore, Experience, and Build the Future of On-Device AI with Nexus AI.**

Nexus AI is a next-generation on-device generative AI platform that lets you run powerful open-source Large Language Models (LLMs) directly on your mobile device. Fully offline, private, and optimized for performance.

---

## ⚡ Why Nexus AI?

- 🧠 Run powerful AI models fully offline
- 🔒 100% private — no data leaves your device
- 🚀 Fast GPU/CPU optimized inference
- 📱 Designed for Android & iOS
- 🤖 Supports text + vision AI models

---

## 🌟 Core Features

### 💬 AI Chat (Smart Mode)
Engage in natural conversations with advanced LLMs. Supports multi-turn memory and contextual reasoning.

---

### 👁️ Ask Image (Vision AI)
Analyze images, detect objects, and get smart descriptions using on-device vision models.

---

### 🎙️ Audio Intelligence
Convert voice to text and understand audio input using lightweight AI models.

---

### 🧪 Prompt Lab
Experiment with AI prompts and fine-tune model behavior with parameters like temperature and top-k.

---

### ⚙️ Model Manager
Download, switch, and manage multiple AI models easily inside the app.

---

### 🔥 Mobile AI Actions
Enable offline smart actions and automation powered by lightweight function-tuned models.

---

### 🧠 Tiny AI Modules
Run ultra-light models optimized for low-end devices with fast response time.

---

## 🏁 Get Started

### Requirements
- Android 12+ / iOS 17+

### Installation
- Download from official app store (coming soon)
- Or install APK from latest releases

---

## 🛠️ Technology Stack

- Google AI Edge Runtime
- LiteRT / MediaPipe LLM Inference
- Hugging Face Model Hub Integration
- On-device GPU acceleration

---

## 🔐 Privacy First

Nexus AI runs completely offline.  
No prompts, images, or voice data are sent to any server.

---

## 📦 Model Support

Nexus AI supports multiple open-source models:

- Gemma 3n (E2B / E4B)
- Gemma 3 1B (Lightweight)
- Qwen 2.5 series
- DeepSeek distilled models
